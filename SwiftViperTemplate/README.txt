
Instrucciones para crear plantilla:


- En el finder, buscar la carpeta "Biblioteca" (Está oculta, presiona " shit + cmd + . " Para verla)
- Dentro del folder "Biblioteca", ir a:
    Developer -> Xcode -> Templates
- Dentro de "Templates", crear carpeta "Design Patterns" (Puedes usar el nombre que desees, este es el que aparecerá como sección al crear nuevo archivo en Xcode)
- Dentro de esta carpeta, colocar el "ViperModule.xctemplate"

Para crearlo:
- En Xcode, primero tienes que crear un grupo ("New Group") con el nombre de tu VIPER
- Posteriormente, dar click secundario en ese nuevo grupo, seleccionas la opción "New File", buscas la sección "Design Patterns" (o el nombre que le pusiste) y seleccionas "VIPER", al crearlo te pedirá el nombre del viper que es el mismo que le pusiste al grupo
